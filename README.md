configs
=======

Some configuration files from my $HOME. Every software configuration will reside in each subdir.
See the file naming pattern inside.

gitconfig
---------

```
$HOME/.gitconfig
$LOCALREPO/.git/config
```

emacs.d
-------

$HOME/.emacs.d/init.el for emacs (currently 24.0.97)

muttrc
------

```
$HOME/.muttrc
```
